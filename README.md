# Jaspreet Kaur — Portfolio Website

A modern, responsive, and interactive personal portfolio website built to showcase my experience, projects, and skills as an AI & Software Engineer.

**Live Demo:** [https://jaspreetkaurai.netlify.app/#]

## 🌟 Features

- **Modern UI/UX:** Sleek, dark-themed design with custom gradients, smooth scrolling, and dynamic typography.
- **Interactive Elements:** Particle canvas background, revealing animations, and a typing effect.
- **Responsive Layout:** fully functional on desktop, tablet, and mobile devices.
- **Dynamic Content Filtering:** Filterable sections for both "Work Experience" and "Featured Projects".
- **Semantic HTML & Accessibility:** Uses proper landmark elements, ARIA labels, and skip-to-content links.
- **SEO Optimized:** Includes meta tags, Open Graph tags, Twitter cards, and JSON-LD structured data.
- **No Frameworks:** Built entirely with vanilla HTML5, CSS3, and JavaScript for maximum performance.

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** CSS3 (Custom Variables, Flexbox, CSS Grid, Media Queries, Keyframes animations)
- **Logic & Interactivity:** Vanilla JavaScript (ES6)
- **Fonts:** Syne, Inter, and JetBrains Mono (from Google Fonts)
- **Assets/Icons:** Inline SVGs for fast loading
